About This Site

This site is built as a static front-end project. The design uses CSS grid, flexbox, gradients, shadows, pseudo-elements, and JavaScript rendering.

To make this use live osu! data, you would connect the leaderboard to a small backend server that requests osu! API data and sends only safe public results to this website.

Made for learning HTML, CSS, and JavaScript

No official affiliation with osu! or The Elder Scrolls.
